# ✂️ Edu Barber — Site Oficial

Site institucional da **Edu Barber**, barbearia localizada no bairro Tatuquara, em Curitiba. Projeto desenvolvido do zero com HTML, CSS e JavaScript puro.

---

## 📋 Sobre o Projeto

Site criado para apresentar a barbearia, facilitar o agendamento de horários e exibir informações de contato e localização. O projeto foi desenvolvido de forma progressiva, com foco em aprendizado prático de desenvolvimento front-end.

---

## 🗂️ Páginas

- **index.html** — Página principal com hero, serviços e seção de agendamento
- **sobre.html** — Página "Sobre Nós" com história da barbearia, equipe e localização

---

## ✅ Funcionalidades

- Animações de entrada com `IntersectionObserver` (scroll-triggered)
- Animações CSS com `@keyframes` ao carregar a página
- Botão de agendamento com link direto para o WhatsApp
- Mapa do Google Maps embutido com localização da barbearia
- Links de navegação no footer com âncoras (`#sobre`, `#equipe`)
- Link para Instagram (`@edu_barber041`)
- Layout responsivo com Flexbox
- Ícones via Ionic Icons
- Navegação suave com `scroll-behavior: smooth`

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, animações, keyframes, transitions)
- JavaScript (IntersectionObserver, DOM manipulation)
- [Ionic Icons](https://ionic.io/ionicons)
- Google Maps Embed API

## 📖 O que foi Aprendido no Projeto

Este projeto foi desenvolvido durante um processo de aprendizado ativo de front-end. Os principais conceitos praticados foram:

- **Box model e margens** — diferença entre `margin`, `padding` e `gap`, colapso de margens
- **Flexbox** — alinhamento de elementos com `align-items`, `justify-content` e `gap`
- **Posicionamento CSS** — uso de `position: relative` para evitar quebras de layout
- **Animações CSS** — `@keyframes`, `transition`, `transform: translateY`, `opacity`
- **IntersectionObserver** — animações ativadas pelo scroll com classes dinâmicas
- **Âncoras HTML** — navegação entre seções com `href="#id"`
- **Git básico** — fluxo de `add`, `commit` e `push` para o GitHub
